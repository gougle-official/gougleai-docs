Welcome to Gougle AI API's documentation!
===================================

**Gougle AI API** (/ˈɡuːɡᵊl eɪ-aɪ eɪ-piː-aɪ/) is a Python library that provide access
to Gougle's AI model trought an API.
It pulls data from the `Gougle AI services <https://ai.withgougle.cf/>`_
and offers a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`install <usage:installation>` the project.

.. note::
   This project is under active development.
